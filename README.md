# Ray Tracing in One Weekend - MoonBit実装

[Ray Tracing in One Weekend](https://raytracing.github.io/)のチュートリアルを[MoonBit](https://www.moonbitlang.com/)プログラミング言語で実装したものです。

## 環境

- [MoonBit](https://www.moonbitlang.com/) 0.1.20250801

## 使用方法

### ビルド

```bash
make build
```

### レンダリング実行

```bash
make run
```

### 画像表示

```bash
make view
```
